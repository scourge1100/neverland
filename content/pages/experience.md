# Experience

## Backend Platform

여러 서비스가 공유하는 회원, 인증, 권한과 운영 기능을 Spring Boot 기반 Core 플랫폼으로 설계하고 개발했다. 서비스 타입으로 정책 범위를 분리하고 PostgreSQL, JPA와 Flyway로 데이터 변경을 관리한다.

## Mobile Product

React Native와 Expo로 심판 업무 앱을 개발하고 Android APK와 OTA 업데이트를 운영한다. 역할별 배정, 공지, 결과 보고, 정산과 푸시 알림처럼 실제 업무가 이어지는 흐름을 API와 함께 구현했다.

## Full-stack Web

Next.js와 Spring Boot로 구성된 테일러샵 팀 프로젝트에 풀스택으로 참여했다. 사용자 화면, 인증과 데이터 API의 책임을 나누고 연결하는 경험을 쌓았다.

## Operations and Automation

기능 구현 이후의 배포, 데이터 마이그레이션, 로그 진단, 백업과 복구 가능성을 함께 본다. 반복되는 확인과 배포 작업은 스크립트와 CI/CD로 옮기고, 문제를 다시 재현할 수 있도록 운영 근거를 문서화한다.

## Working principles

- 실제 코드, 데이터, 로그와 실행 결과를 기준으로 판단한다.
- 서비스 경계와 변경의 영향을 먼저 확인한다.
- 실패를 관찰할 수 있도록 로그와 진단 정보를 남긴다.
- 배포 성공이 아니라 실제 사용 가능한 상태까지 검증한다.

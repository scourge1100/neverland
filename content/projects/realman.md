# RealMan Tailor

## 개요

결혼, 면접과 중요한 일정에 정장이 필요한 사용자가 전문 테일러를 찾고 상담·예약할 수 있도록 기획한 팀 프로젝트다.

## 서비스 방향

- 테일러샵과 상품 탐색
- 원하는 테일러와 예약 연결
- 게시물 작성과 상세 조회
- 사용자 인증과 서비스 데이터 관리

## 역할

프론트엔드와 백엔드를 함께 다루는 풀스택 팀원으로 참여했다. 화면과 API가 나뉜 구조에서 사용자 흐름과 데이터 계약을 연결하는 경험을 쌓았다.

## 기술

### Frontend

- Next.js 15
- React 19 / TypeScript
- Tailwind CSS
- React Hook Form / Zod
- Zustand

### Backend

- Java / Spring Boot
- Spring Security / OAuth2 / JWT
- Spring Data JPA / MyBatis
- PostgreSQL

## 배운 점

팀 단위로 화면과 API 계약을 맞추는 과정, 인증이 필요한 사용자 흐름, 프론트엔드와 백엔드의 책임을 나누는 기준을 실제 프로젝트 구조에서 경험했다.
